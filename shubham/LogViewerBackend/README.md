
# Project Title:LogViewer

This Project is created for the developer to find the Exception and Errors in a docker containers and resolve them fastly.

## Features

- Get Running Container
- Error Logs 


## Technology
- React
- Spring Boot

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

then git clone Rest Api of Spring Boot LogViewer

'''
Start Rest Api Web Server

'''
## Running Tests

To run tests, run the following command

```bash
  npm run test
```
